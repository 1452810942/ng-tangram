# Angular 8+ Components

[![npm version](https://badge.fury.io/js/%40ng-tangram%2Fcomponents.svg)](https://www.npmjs.com/@ng-tangram/components)

## 安装

### npm
`npm install @ng-tangram/components --save` 

### yarn
`yarn add @ng-tangram/components`
