## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 属性：标题 | `string` | - |
| color | 属性：颜色 | `primary`、 `secondry`、 `success`、 `warning`、 `alert` | `primary` |
| size | 属性：提示框大小 | `small`、 `medium`、`large` | `medium` |
| closable | 属性：是否显示关闭按钮 | `boolean` | - |
| close | 事件：关闭时触发 | `function` | - |
