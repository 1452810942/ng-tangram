## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 属性：标题 | `string` | - |
| color | 属性：颜色 | `primary`、 `secondry`、 `success`、 `warning`、 `alert` | `primary` |
| size | 属性：尺寸 | `small`、 `medium`、`large` | `medium` |
| closable | 属性：可关闭的提示框 | `boolean` | `false` |
| closed | 事件：关闭事件 | `function` | - |
