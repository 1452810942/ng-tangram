## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | 属性：头像大小 | `number`、 `small`、 `defulat`、`large`  | `defulat`|
| shape | 属性：形状（方形、圆形） |  `circle`、 `square` | `circle` |
| thumbnail | 属性：缩略图地址  | string | 无 |
| src | 属性：原图地址（用于显示大图）  | string | 无  |
