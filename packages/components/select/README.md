## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| placeholder | 属性：提示语 | `string` | - |
| multiple | 属性：多选模式 | `boolean` | `false` |
| disabled | 属性：不可用状态 | `boolean` | `false` |
| required | 属性：必须输入标识（只在模板驱动表单有效） | `boolean` | `false` |
| compareWith | 属性：值比较器 | `function` | - |
| selectionChange | 事件：选中值改变事件 | `function` | - |
